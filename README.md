# Social Media Analytics Dashboard (Excel)

## Project Overview
This project analyzes social media performance using Excel. The dataset includes engagement metrics such as likes, comments, shares, impressions, and engagement rate.

## Objective
To identify top-performing platforms and content types using data analysis and visualization.

## Dataset
The dataset contains 100 social media posts with the following columns:

- Post_ID
- Platform
- Post_Date
- Content_Type
- Likes
- Comments
- Shares
- Impressions
- Engagement_Rate

## Dashboard Features
- Platform engagement comparison
- Content type performance analysis
- Likes trend visualization
- Top engagement metrics

## Tools Used
- Microsoft Excel
- Data Analysis
- Data Visualization

## Insights
The dashboard helps understand which platform and content type generate the highest engagement.
